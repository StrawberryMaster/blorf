This is **blorf**, a fork of [EnceladusCat](https://github.com/enceladuscat)'s [STORM_INC/TCM](https://enceladuscat.github.io/Storm_Inc/TCM.html), an advanced tropical cyclone dynamics engine. This is a somewhat more modern rewrite of the original TCM engine, with a focus on modernizing the codebase but especially improving performance on devices in general, alongside translating from Chinese to English and paving the way for easier updates and maintenance of this code.

Per the original TCM engine's about page, we must also note the original inspiration was from ProtoManly on Minecraft/Roblox (2016).
